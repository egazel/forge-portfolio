# **App Name**: Elie Gazel Portfolio

## Core Features:

- Homepage Showcase: Display a grid of recent projects with prominent imagery and titles to showcase visual work.
- Project Details: Dedicated pages for each project featuring detailed descriptions, videos, images, and technologies used.
- Interactive 3D Model Viewer: Allow potential employers to directly interact with 3D models on the site (powered by Three.js or similar).
- Blog: Showcase knowledge, expertise, and thoughts on recent advancements within the gaming and 3D animation industry.
- Contact Form with Validation: Implement a form for easy communication with potential clients or employers.
- AI Portfolio Curator Tool: AI tool reviews project content and suggests tagging it with skills like 'python' or 'Maya API'.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5), providing a high-tech and creative feel.
- Background color: Very dark gray (#212121), creating a sophisticated backdrop.
- Accent color: Electric Purple (#BE71F0), drawing attention to key interactive elements.
- Body font: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif
- Use minimalist line icons for navigation and key feature representations, with subtle hover animations.
- Employ a grid-based layout with generous white space to ensure clarity and focus on project showcases.
- Incorporate smooth transitions and subtle animations for loading, scrolling, and interactive elements to provide a polished user experience.
